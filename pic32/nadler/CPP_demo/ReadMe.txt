For compilation with CodeSourcery 4.80 C++.

To run example using MPLAB and the USB Starter Board
1) Connect USB starter board to PC with MPLAB installed.
2) Start MPLAB
3) Select Menu >Debugger>Select Tool>PIC32 Starter Kit
4) You should see "[Debugger] Starter Kit Found
5) Select Menu >File>Import... provide path to CPP_Demo.hex
6) Click on the "Program All Memories" icon
7) Click on the "Run" icon

You should now see LEDs 1 and 2 blinking...
